# SUDO物业管理系统

本项目基于vue-element-admin和element-ui制作，是社达SUDO前端部分的物业端，目前提供信息审核和通知推送的功能。

使用说明：
```shell
npm install  // 安装依赖库
npm run dev  // 本地运行
npm run build:prod  // 打包
```

运行端口：9527
